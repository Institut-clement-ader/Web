Gérer les demandes ZRR¶

Cette page fait appel au snippet consulter-demandes-zrr. La page met en place un tableau dans lequel on affiche les demandes ainsi qu’un bouton accepter et refuser. On peut voir les deux if à la fin du code définir le fonctionnement des deux boutons.

Dépôt des dossier ZRR¶

Cette page fait appel au snippet depot-zrr . Dans le code du snippet on peut voir au départ l’attribution de valeur à toutes les variables.

Les if suivant permettent d’envoyer un mail avec le fichier zip déposé à un membre de l’administration. Enfin on peut retrouver tout le formulaire dans la suite du code contenant le select avec de nombreuses options ainsi que les input.
