===
ZRR
===

Gérer les demandes ZRR
======================

Cette page fait appel au snippet *consulter-demandes-zrr*. La page met en place un tableau
dans lequel on affiche les demandes ainsi qu'un bouton accepter et refuser.
On peut voir les deux if à la fin du code définir le fonctionnement des deux boutons.




Dépôt des dossier ZRR
=====================

Cette page fait appel au snippet *depot-zrr* . Dans le code du snippet on peut
voir au départ l'attribution de valeur à toutes les variables.

Les if suivant permettent d'envoyer un mail avec le fichier zip déposé à un membre de l'administration.
Enfin on peut retrouver tout le formulaire dans la suite du code contenant le select avec de nombreuses options
ainsi que les input. 


Mes demandes ZRR
================

Cette page fait appel au snippet *mes-demandes-zrr*

Pour l'affichage on peut voir dans le code la création du tableau en première partie.
Ensuite on retrouve le code pour mettre à jour le numéro de dossier.
Enfin la dernière partie du code est basée sur le bouton accepter et refuser

.. note::
    Le bouton accepter envoie un mail à 4 personnes, on peut retrouver ces mails dans le code

Démarches à suivre
==================

Cette page n'appelle aucun snippet et affiche uniquement du texte ainsi que des liens. Vous pouvez les retrouver directement
sur le gestionnaire WordPress.
